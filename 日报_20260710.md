## 一、Compose 项目功能完成情况

### 1.1 之前已完成的存量功能

以下功能在本次周报周期之前已实现完毕：

| 序号 | 功能模块 | 说明 |
|------|----------|------|
| 1 | 播放/暂停控制 | 基本的播放/暂停切换，使用 MediaPlayer 实现 |
| 2 | 上一首/下一首 | 支持顺序切换歌曲，根据播放模式决定下一首 |

### 1.2 上周新增功能（7月7日 ~ 7月10日）

#### 功能一：播放速度控制

**功能实现**：支持 0.5x、0.75x、1.0x、1.25x、1.5x 五档倍速播放。

**实现方式**：
- 在 `MusicViewModel.kt` 中新增 `changePlaybackSpeed(speed: Float)` 方法（第 194~204 行）
- 使用 `MediaPlayer.playbackParams.setSpeed()` 设置播放速率
- 通过 `coerceIn(0.5f, 2.0f)` 限制速度范围
- 切换歌曲时自动重置为 1 倍速（`playSong()` 第 106、110 行）
- UI 层在 `PlayerDetailScreen.kt` 中新增倍速按钮（第 248~263 行），点击循环切换五档速度

**涉及文件**：
- `MusicViewModel.kt`（新增 `changePlaybackSpeed` 方法）
- `PlayerDetailScreen.kt`（新增倍速切换按钮）
- `MusicUiState.kt`（新增 `playbackSpeed: Float = 1f` 字段）

#### 功能二：快进/快退 10 秒

**功能实现**：支持一键快进 10 秒、快退 10 秒。

**实现方式**：
- 在 `MusicViewModel.kt` 中新增 `seekForward10s()` 方法（第 249~264 行）和 `seekBackward10s()` 方法（第 269~283 行）
- 快进逻辑：`newPosition = (currentPosition + 10000).coerceAtMost(duration)`，不超过总时长
- 快退逻辑：`newPosition = (currentPosition - 10000).coerceAtLeast(0)`，不低于 0
- 跳转后自动调用 `updateLyric()` 刷新歌词显示
- 使用 `isUserSeeking` 标志防止进度循环覆盖
- UI 层在 `PlayerDetailScreen.kt` 中添加快退按钮（第 267~276 行）和快进按钮（第 325~334 行），分别使用 `bg_backward.png` 和 `bg_forward.png` 图标
- `MainActivity.kt` 中绑定回调（第 43~44 行）

**涉及文件**：
- `MusicViewModel.kt`（新增 `seekForward10s`、`seekBackward10s` 方法）
- `PlayerDetailScreen.kt`（新增快进/快退图标按钮）
- `MainActivity.kt`（新增 `onSeekForwardClick`、`onSeekBackwardClick` 绑定）

#### 功能三：进度跳转

**功能实现**：支持两种精准跳转方式——拖动进度条跳转和点击歌词跳转。

**实现方式**：

**3.1 拖动进度条跳转**（已有 UI，上周完善边界处理）：
- `PlayerDetailScreen.kt` 第 196~202 行：Slider 组件，拖动时触发 `onProgressChange`
- `MusicViewModel.kt` 第 206~224 行：`onProgressChange()` 更新 UI 位置并同步歌词；`onSeekFinished()` 松手后 `seekTo` 生效

**3.2 点击歌词跳转**（上周新增）：
- `MusicViewModel.kt` 中新增 `onLyricClick(lyricText: String)` 方法（第 226~244 行）
- 根据点击的歌词文本在 `lyricLines` 中查找对应时间戳
- 调用 `mediaPlayer.seekTo(lyricLine.timeMs)` 跳转到该歌词位置
- 自动同步歌词并更新 UI 状态
- 如果当前未播放，自动开始播放
- UI 层 `PlayerDetailScreen.kt` 第 172~174 行：歌词 Text 添加 `clickable` 修饰符

**涉及文件**：
- `MusicViewModel.kt`（新增 `onLyricClick` 方法）
- `PlayerDetailScreen.kt`（歌词项添加点击事件）
- `MainActivity.kt`（新增 `onLyricClick` 绑定）
